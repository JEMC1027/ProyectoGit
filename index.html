¿Que es?

El DOM (Document Object Model) es una representación en memoria de un documento HTML o XML. Es una interfaz que permite a los lenguajes de programación, especialmente JavaScript, interactuar con los contenidos y la estructura de una página web de manera dinámica. A través del DOM, los elementos HTML se convierten en objetos que pueden ser modificados, eliminados o creados.

Caracteristicas

Estructura Jerárquica:

El DOM organiza el documento como un árbol de nodos, donde cada elemento HTML (como div, p, etc.), atributo y contenido de texto es representado como un nodo dentro de esa estructura. El nodo raíz del árbol es el documento completo.

Interactividad

El DOM permite la manipulación dinámica de una página web. A través de JavaScript, se pueden cambiar elementos, agregar nuevos contenidos, modificar atributos, o reaccionar a eventos (como clics de usuario) sin necesidad de recargar la página

Acceso a Elementos

JavaScript puede acceder a los elementos del DOM utilizando métodos como getElementById, getElementsByClassName, querySelector, etc. Esto permite obtener o modificar valores de los elementos de la página.

Independencia del lenguaje

Aunque el DOM es generalmente manipulado con JavaScript en los navegadores, es un estándar que puede ser usado por diferentes lenguajes de programación

DOM en Java

Java ofrece una API conocida como Java API for XML Processing (JAXP) que incluye un conjunto de herramientas y clases para trabajar con documentos XML. DOM es uno de los métodos de manipulación de documentos XML que permite cargar un documento XML en memoria y tratarlo como un árbol de nodos (de elementos, atributos, texto, etc.).
